# 📖 Overview

`claude-code-notebookllm-cheat-code` transforms Google NotebookLM into a programmable ingestion engine for AI-native research and production systems.

Instead of:
- ❌ downloading YouTube videos
- ❌ parsing broken subtitles
- ❌ fighting IP bans
- ❌ maintaining fragile scraping stacks

This project delegates the heavy lifting to Google's own ingestion and transcript-cleaning infrastructure, then extracts the structured outputs locally through `notebooklm-py`.

The result is a terminal-first workflow for:
- 🧾 transcript extraction
- 🎙️ podcast synthesis
- 🧠 structured research generation
- 🤖 Claude Code context injection
- 🔓 hidden artifact retrieval
